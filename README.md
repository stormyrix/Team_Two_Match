# Платформа текстовых трансляций спортивных событий "Матч"
## Функционал
- текстовые трансляции матчей
- автоматическое  обновление счёта
- обсуждение событий
- получаение уведомлений о важных моментах

## Использованный стек.
- Go+gin+gorm
- postgreSQL
- Kafka
- Docker
- Redis
